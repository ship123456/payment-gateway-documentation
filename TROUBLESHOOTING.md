## Introduction

This guide provides solutions to common issues that users may encounter while using the Payment Gateway Dashboard. Follow the recommended solutions before contacting the support team.

## Browser & Connectivity Issues

| Issue | Possible Solution |
|--------|-------------------|
| Application is not loading | Refresh the page and verify your internet connection. |
| Blank screen after opening the application | Clear the browser cache and reload the application. |
| Slow application performance | Close unnecessary browser tabs and refresh the application. |
| Browser not supported | Use the latest version of Google Chrome, Microsoft Edge, Mozilla Firefox, or Safari. |
| Changes not saved | Verify that Local Storage is enabled in your browser. |

> **Note:** For the best experience, use the latest version of Google Chrome.

## Frequently Asked Questions

### Can I edit a payment after it has been created?

No. Existing payment records are read-only. Create a new payment if changes are required.

### Why can't I see my payment in Payment History?

Refresh the Payment History page and verify that the payment was created successfully.

### Why are webhook logs not appearing?

Ensure that webhooks are enabled before creating a payment. Only generated webhook events are recorded.

### Can I update my profile information?

Yes. Navigate to **Profile**, update the required fields, and click **Save Changes**.

### How do I change the application theme?

Open **Settings** and choose either the **Light** or **Dark** theme.

### Why are my changes not saved after refreshing the browser?

The application stores data locally using browser Local Storage. Clearing browser data or using another browser or device may remove locally stored information.

### Can I upload attachments when contacting support?

Yes. The **Contact Support** form supports file attachments.

### What should I do if a feature is not working?

Refresh the application, verify your internet connection, and ensure all required information has been entered. If the issue persists, contact the support team.

### Why can't I copy the Webhook URL?

Ensure your browser allows clipboard access and click **Copy URL** again.
